## About The Project

A simple webpage for displaying the current lab configuration for easier navigation and editing.

### Built With

## Getting Started

### Prerequisites

Install the following requirements on your system:

* Python 3.10+

### Installation

1. Clone the repo
2. Create virtual environment
3. Install required packages from requirements.txt
4. Run Flask command

### Usage

* Create admin account
* Create rack and server
